ACL Prospect Enrichment v6

This is a cron-based enrichment service. No UI needed.

Cron endpoint: /api/cron/enrich-queue

Data Sources

  • IPEDS (enrollment, acceptance rates)
  • Semantic Scholar (citations, papers)
  • OpenAlex (academic profiles)
  • University RSS feeds (news)
  • Direct URL probing (bio pages)
  • Google Custom Search (fallback)
